# Card Universal con Slots y Borde Personalizado

- Header, Content y Footer pueden ser pasados como templates.
- `borderWidth` permite definir el grosor del borde (en px).
- Compatible con `interactive` y `disabled`.

---

## Props

| Prop          | Tipo                         | Descripción                          |
| ------------- | ---------------------------- | ------------------------------------ |
| `title`       | `string`                     | Título principal.                    |
| `subtitle`    | `string`                     | Subtítulo.                           |
| `image`       | `string`                     | Imagen opcional.                     |
| `layout`      | 'vertical' \| 'horizontal'   | Distribución.                        |
| `align`       | 'start' \| 'center' \| 'end' | Alineación.                          |
| `size`        | 'sm' \| 'md' \| 'lg'         | Tamaño.                              |
| `interactive` | `boolean`                    | Habilita hover y foco.               |
| `disabled`    | `boolean`                    | Bloquea la card.                     |
| `bordered`    | `boolean`                    | Añade borde.                         |
| `borderWidth` | `number`                     | Ancho del borde en px.               |
| `accent`      | `boolean`                    | Aplica color de acento al borde.     |
| `header`      | `TemplateRef<any>`           | Template personalizado para header.  |
| `content`     | `TemplateRef<any>`           | Template personalizado para content. |
| `footer`      | `TemplateRef<any>`           | Template personalizado para footer.  |
