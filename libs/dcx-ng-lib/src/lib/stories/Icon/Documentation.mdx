# Icon

The Icon component is a flexible UI element that uses Bootstrap Icons for consistent and modern iconography.
You can find the name of your icon on the [Bootstrap Icons](https://icons.getbootstrap.com/) website.

---

## Example

<Canvas of={IconClassBasedStories.MediumCompact} />

You can customize:

- **name**: the Bootstrap icon name (e.g., 'heart-fill', 'star-fill', 'search')
- **color**: the icon color (hex or named color)
- **size**: s (small), m (medium), l (large), xl (extra-large)
- **spacing**: none, compact, spacious

---

## Usage Example

```html
<dcx-ng-icon
  [name]="'search'"
  [color]="'#27ae60'"
  [size]="'m'"
  [spacing]="'spacious'"
>
</dcx-ng-icon>
```

## Popular Icons

- **Navigation**: `house-fill`, `arrow-left`, `arrow-right`, `chevron-up`, `chevron-down`
- **Status**: `check-circle`, `x-circle`, `exclamation-circle`, `info-circle`, `question-circle`
- **UI**: `search`, `bell`, `gear`, `download`, `upload`, `trash`, `edit`, `eye`, `eye-slash`
- **Ratings**: `star`, `star-fill`, `heart`, `heart-fill`

For a complete list of available icons, visit [Bootstrap Icons](https://icons.getbootstrap.com/)
