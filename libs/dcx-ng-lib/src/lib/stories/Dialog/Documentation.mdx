# Dialog

`dcx-ng-dialog` muestra un cuadro modal con título, contenido y pie opcional.

## Inputs

- `title: string` - Título del diálogo
- `visible: boolean` - Controla la visibilidad del diálogo

## Outputs

- `(onClose): void` - Se emite cuando se cierra el diálogo

## Uso

### Con contenido simple

```html
<dcx-ng-dialog [title]="'Título'" [visible]="visible" (onClose)="handleClose()">
  <div dialog-body>Contenido</div>
  <div dialog-footer>
    <button (click)="handleClose()">Cerrar</button>
  </div>
</dcx-ng-dialog>
```

### Con templates

```html
<dcx-ng-dialog [title]="'Título'" [visible]="visible" (onClose)="handleClose()">
  <ng-template #dialogBody>
    <p>Contenido con template</p>
  </ng-template>
  <ng-template #dialogFooter>
    <button (click)="handleAccept()">Aceptar</button>
  </ng-template>
</dcx-ng-dialog>
```
