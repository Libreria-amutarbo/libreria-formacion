# Dialog

`dcx-ng-dialog` muestra un cuadro modal con título, contenido y pie opcional.

- **Inputs**:
  - `title: string`
  - `visible: boolean`
- **Outputs**:
  - `(onClose): void`

## Uso

```html
<dcx-ng-dialog [title]="'Título'" [visible]="true" (onClose)="handleClose()">
  <div dialog-body>Contenido</div>
  <div dialog-footer>
    <button (click)="handleClose()">Cerrar</button>
  </div>
</dcx-ng-dialog>
```
