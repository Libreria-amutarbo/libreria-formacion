# Chip Component

The Chip component provides a compact way to display information, labels, or selectable elements with built-in theming and interactive capabilities.

## Features

- **Theme colors** - Supports 8 predefined colors from the design system
- **Icons** - Optional Material Design icons
- **Removable** - Optional delete functionality with event handling
- **Accessible** - ARIA labels and keyboard navigation support

## Basic Usage

```html
<dcx-ng-chip label="Basic Chip"></dcx-ng-chip>
```

## With Icon

```html
<dcx-ng-chip 
  label="Settings" 
  icon="settings" 
  color="primary">
</dcx-ng-chip>
```

## Removable Chip

```html
<dcx-ng-chip 
  label="Removable Tag" 
  color="warning" 
  [removable]="true"
  (onRemove)="handleChipRemove()">
</dcx-ng-chip>
```

### Theme Colors

Available color options: `primary`, `secondary`, `success`, `warning`, `error`, `info`, `gray`, `gray-light`

## Component Structure

The component automatically determines its type based on inputs:
- **Text only** - Basic chip with label
- **With icon** - Icon + label combination  
- **Removable** - Includes delete button (Ã—)
- **Icon + Removable** - Full-featured chip

## Accessibility

- Uses appropriate `aria-label` attributes
- Delete button includes descriptive text for screen readers
- Proper semantic HTML structure
- Keyboard navigation support
