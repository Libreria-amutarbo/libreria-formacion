# List

`dcx-ng-list` es un componente **mínimo** para renderizar una lista a partir de un array.

- API: un único `@Input() items: Array<string | number>`.
- Render: usa el nuevo control flow de Angular con `@for`.
- Sin selección, sin eventos, sin estilos complejos.

## Uso básico

```html
<dcx-ng-list [items]="['Alpha', 'Beta', 'Gamma']"></dcx-ng-list>
```
