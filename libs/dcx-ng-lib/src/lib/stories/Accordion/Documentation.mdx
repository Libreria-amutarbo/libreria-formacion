# Accordion

`dcx-ng-accordion` es un contenedor de elementos expandibles/colapsables usado para organizar contenido de forma compacta y accesible.

- **API mínima**: `@Input() items: DcxNgAccordionItem[]`
- **Múltiples items abiertos**: Permite tener varios items expandidos simultáneamente con `[closeOthers]="false"`
- **Items deshabilitados**: Algunos items pueden estar bloqueados
- **Contenido deshabilitado**: El contenido de un item puede estar deshabilitado visualmente
- **Animaciones suaves**: Transiciones elegantes al expandir/colapsar (configurable)
- **Accesibilidad**: Atributos ARIA para lectores de pantalla

## Uso

```html
<dcx-ng-accordion
  [items]="items"
  transition="smooth"
  [closeOthers]="true"
  (itemToggled)="onItemToggled($event)">
</dcx-ng-accordion>
```

Propiedades del objeto `items`:

```typescript
export interface DcxNgAccordionItem {
  id: string;
  title: string;
  content: string;
  disabled?: boolean;
  expanded?: boolean;
  icon?: string;
  disabledContent?: boolean;
}
```
