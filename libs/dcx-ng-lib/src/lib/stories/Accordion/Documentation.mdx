# Accordion

`dcx-ng-accordion` es un contenedor de elementos expandibles/colapsables usado para organizar contenido de forma compacta y accesible.

- **API mínima**: `@Input() items: AccordionItem[]`
- **Múltiples items abiertos**: Permite tener varios items expandidos simultáneamente
- **Items deshabilitados**: Algunos items pueden estar bloqueados
- **Animaciones suaves**: Transiciones elegantes al expandir/colapsar
- **Accesibilidad**: Atributos ARIA para lectores de pantalla

## Uso

```html
<dcx-ng-accordion
  [items]="[
    { id: '1', title: 'Item 1', content: 'Contenido 1', disabled: false },
    { id: '2', title: 'Item 2', content: 'Contenido 2', disabled: false },
    { id: '3', title: 'Item 3', content: 'Contenido 3', disabled: true }
  ]"
  (itemToggled)="onItemToggled($event)">
</dcx-ng-accordion>
```
