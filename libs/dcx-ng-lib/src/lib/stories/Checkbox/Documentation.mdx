# Checkbox

El componente `dcx-ng-checkbox` permite seleccionar valores booleanos en formularios y vistas, mostrando un check (✔) usando estilos SCSS.

Este componente está desarrollado con **Angular Signals** para mejor rendimiento y reactividad.

## Versión Class-Based

La versión class-based usa estilos mínimos y variables CSS para flexibilidad y personalización.

## Personalización

Puedes personalizar:

- **label**: texto mostrado junto al checkbox
- **color**: primary, accent, error o color personalizado (hex, rgb)
- **checked**: estado marcado/desmarcado
- **disabled**: deshabilita la interacción
- **errorMessage**: mensaje de error personalizado
- **size**: tamaño del checkbox ('s' | 'm' | 'l' | 'xl' | 'auto')
  - **s**: 1rem (16px)
  - **m**: 1.5rem (24px) - por defecto
  - **l**: 2rem (32px)
  - **xl**: 3rem (48px)
  - **auto**: tamaño automático

### Para Grupos:

- **options**: array de opciones `CheckboxOption[]`
- **selectedValues**: array de valores seleccionados
- **multiple**: permite selección múltiple (true) o única (false)
- **groupLabel**: label para el grupo

## Ejemplo de Uso

### Checkbox Individual

```html
<dcx-ng-checkbox
  label="Acepto los términos"
  color="primary"
  size="m"
  [checked]="checked"
  (checkedChange)="checked = $event"
  errorMessage="Este campo es obligatorio"
></dcx-ng-checkbox>
```

### Checkbox con Tamaño Personalizado

```html
<dcx-ng-checkbox
  label="Checkbox grande"
  color="accent"
  size="xl"
  [checked]="checked"
  (checkedChange)="checked = $event"
></dcx-ng-checkbox>
```

### Grupo con Selección Múltiple

```html
<dcx-ng-checkbox
  groupLabel="Selecciona tus intereses"
  [options]="options"
  [selectedValues]="selectedValues"
  [multiple]="true"
  size="m"
  color="primary"
  (selectionChange)="handleSelection($event)"
></dcx-ng-checkbox>
```

### Grupo con Selección Única (tipo Radio)

```html
<dcx-ng-checkbox
  groupLabel="Selecciona tu favorito"
  [options]="options"
  [selectedValues]="selectedValues"
  [multiple]="false"
  size="m"
  color="accent"
  (selectionChange)="handleSelection($event)"
></dcx-ng-checkbox>
```

## Interface CheckboxOption

```typescript
interface CheckboxOption {
  value: string;
  label: string;
  disabled?: boolean;
}
```
