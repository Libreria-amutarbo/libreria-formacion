# Input Component

The `dcx-ng-input` component is a versatile form utility designed to capture user input with comprehensive validation and styling options. 

## Overview

This component provides a complete input solution with:
- Multiple input types support
- Size variations
- Validation states
- Accessibility features
- Customizable styling

---
### Class-Based Version

The class-based input implements a design system built with:
- **BEM methodology** for consistent naming conventions
- **SCSS** for maintainable styles
- **CSS custom properties** for flexible theming
- **Responsive design** principles

## Properties

### Input Types
- **text**: Standard text input
- **number**: Numeric input with validation
- **email**: Email input with format validation
- **password**: Password input with hidden characters
- **date**: Date picker input
- **search**: Search input with enhanced styling
- **tel**: Telephone number input
- **url**: URL input with validation

### Sizes
- **small (s)**: Compact size for dense layouts
- **medium (m)**: Standard size for most use cases
- **large (l)**: Larger size for better accessibility
- **extra large (xl)**: Maximum size for prominence

### States
- **disabled**: Prevents user interaction
- **required**: Indicates mandatory field completion

### Content
- **placeholder**: Hint text displayed when empty
- **label**: Descriptive text for the input field

---

## Accessibility

The component includes built-in accessibility features:
- Proper ARIA labels and descriptions
- Keyboard navigation support
- Screen reader compatibility
- Focus management
- Error state announcements

---

## Usage

### Basic Implementation
```html
<dcx-ng-input 
  label="User Name" 
  placeholder="Enter your name..." 
  [disabled]="false" 
  [required]="true"
  [type]="'text'" 
  [size]="'m'" 
  (valueChange)="onValueChange($event)">
</dcx-ng-input>
```
