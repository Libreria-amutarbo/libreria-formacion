# Divider

The `dcx-ng-divider` component is a flexible layout utility for visually separating content.
It supports horizontal and vertical orientations, multiple size presets, solid and dotted styles,
and full color/thickness customization via inputs.

---

## Inputs

| Input | Type | Default | Description |
|---|---|---|---|
| `orientation` | `horizontal \| vertical` | `horizontal` | Direction of the divider line |
| `size` | `s \| m \| l \| xl \| auto` | `auto` | Length of the divider. `auto` fills the parent container |
| `color` | `string` | primary | Any valid CSS color value |
| `thickness` | `number` | `0.25` | Thickness of the line in rem units |
| `type` | `default \| dot` | `default` | Visual style of the line |
| `ariaLabel` | `string` | `'Divider'` | Accessible label for screen readers |

---

## Usage

```html
<dcx-ng-divider
  orientation="horizontal"
  size="m"
  color="#0058ab"
  [thickness]="0.25"
  ariaLabel="Section divider"
></dcx-ng-divider>
```

## Notes

- For **vertical** dividers, the parent container must have a defined height.
- Use `size="auto"` to let the divider fill the available space in its container.
- Use `type="dot"` for a dotted line style.
