# Button

The `dcx-ng-button` component is a flexible, accessible button that supports labels, icons, multiple variants and sizes, and native button types.

---

## Unstyled Version

Use this version when you want full control over the appearance.  
(For demo purposes, this story uses the same component with minimal configuration.)

---

## Class-Based Version

The class-based button uses minimal default styles with CSS variables, BEM naming, and SCSS.

<Canvas of={ButtonClassBasedStories.Primary} />

You can customize:

- **variant**: `primary`, `secondary`, `link`, `icon`
- **size**: `small`, `medium`, `large`, `block`
- **disabled**: disables the button
- **label / ariaLabel**: visible text or accessible name for icon-only buttons
- **iconName**: Material Icons name (e.g., `"save"`, `"settings"`)
- **iconPosition**: `start` | `end`
- **iconSize**: `s` | `m` | `l` | `xl`
- **iconSpacing**: `none` | `compact` | `spacious`
- **iconColor**: hex or CSS color

---

## Usage Example

```html
<!-- Label + start icon -->
<dcx-ng-button
  label="Start Now"
  type="submit"
  variant="primary"
  size="medium"
  iconName="arrow_forward"
  iconPosition="start">
</dcx-ng-button>

<!-- Icon-only (accessible) -->
<dcx-ng-button
  ariaLabel="Open settings"
  variant="icon"
  iconName="settings">
</dcx-ng-button>
