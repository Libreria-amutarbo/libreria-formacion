# Tooltip

The `dcx-ng-tooltip` component is a versatile utility designed to display contextual information through interactive tooltips. It provides accessible, customizable hover messages that enhance user experience by offering helpful hints and additional context without cluttering the interface.

## Configuration Options

### **Position Values**

Determines where the tooltip appears relative to the trigger element

- **`top`**: Tooltip appears above the trigger element
  <Canvas of={TooltipClassBasedStories.TopTooltip} />
- **`bottom`**: Tooltip appears below the trigger element
  <Canvas of={TooltipClassBasedStories.BottomTooltip} />
- **`left`**: Tooltip appears to the left of the trigger element
  <Canvas of={TooltipClassBasedStories.LeftTooltip} />
- **`right`**: Tooltip appears to the right of the trigger element
  <Canvas of={TooltipClassBasedStories.RightTooltip} />

### **Accessibility Features**

- **ARIA Attributes**: Includes `role="tooltip"`, `aria-hidden`, and `aria-live`
- **Keyboard Navigation**: Compatible with screen readers and keyboard navigation
- **Screen Reader Support**: Provides appropriate context for assistive technologies
- **Focus Management**: Proper focus handling for interactive elements

### **Event Handling**

- **Mouse Enter**: Shows tooltip after a brief delay
- **Mouse Leave**: Hides tooltip immediately
  <Canvas of={TooltipClassBasedStories.DefaultTooltip} />
- **Document Click**: Optionally hides tooltip when `hideTooltipOnClick` is enabled
  <Canvas of={TooltipClassBasedStories.HideOnClickTooltip} />
- **Responsive Behavior**: Adapts to touch devices and mobile interactions

---

## Usage Example

```html
<dcx-ng-tooltip
    content="This is my tooltip"
    position="top"
    [hideTooltipOnClick]="false">
  <button>Default</button>
</dcx-ng-tooltip>
```
