# Breadcrumb

Componente para mostrar la jerarquía de navegación.

## Cuándo usar

- Cuando el usuario necesita volver a niveles superiores.
- No usar si la jerarquía es plana.

## Accesibilidad

- Usa `<nav aria-label="Breadcrumb">`.
- El último elemento debe tener `aria-current="page"`.

## Uso

```html

  [items]="[
    { label: 'Home', href: '/' },
    { label: 'Library', href: '/library' },
    { label: 'Data' }
  ]">
</dcx-ng-breadcrumb>
```
