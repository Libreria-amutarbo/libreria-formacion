# Breadcrumb

`dcx-ng-breadcrumb` muestra la ruta de navegación desde la raíz hasta la página actual.

- **API mínima**: `@Input() items: { label: string; href?: string }[]`
- Último ítem se marca como **actual** (`aria-current="page"`).
- Separador por defecto: `/`.
- Semántica accesible: `<nav aria-label="Breadcrumb">` + `<ol role="list">`.

## Uso

```html
<dcx-ng-breadcrumb
  [items]="[
    { label: 'Home', href: '/' },
    { label: 'Library', href: '/library' },
    { label: 'Data' }
  ]">
</dcx-ng-breadcrumb>
```
