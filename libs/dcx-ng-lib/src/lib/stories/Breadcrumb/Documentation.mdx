# Breadcrumb

Componente para mostrar la jerarquía de navegación y la página actual.

## Cuándo usar

- Cuando el usuario necesita entender dónde está dentro de una jerarquía.
- Cuando debe poder volver a niveles superiores de navegación.
- Evitarlo si la jerarquía es plana o hay solo un nivel.

## Accesibilidad

- Usa `<nav aria-label="Breadcrumb">` para anunciar la región.
- El último elemento del breadcrumb se marca con `aria-current="page"`.

## Uso

```html
<dcx-ng-breadcrumb
  [items]="[
    { label: 'Home', href: '/', disabled: false },
    { label: 'Library', href: '/library', disabled: false },
    { label: 'Data', disabled: false }
  ]"
  iconSeparator="chevron-right">
</dcx-ng-breadcrumb>
```
