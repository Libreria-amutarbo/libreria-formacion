Progressbar Component

El componente Progressbar muestra el progreso de una tarea o proceso. Soporta diferentes variantes: barra simple, segmentada, stepper con pasos numerados y stepper con labels.

Loading Segmented

Barra de progreso con segmentos animados, ideal para estados de carga. Configurable con 3, 5 o 10 segmentos.

Progress with Tooltip

Barra de progreso con tooltip mostrando el porcentaje actual.

Simple Progress

Barra de progreso básica con porcentaje mostrado arriba.

Numbered Stepper

Stepper con pasos numerados, mostrando el progreso a través de múltiples etapas.

Checkmark Stepper

Stepper con checkmarks para indicar pasos completados.

Process Stepper

Stepper con labels descriptivos para procesos de múltiples pasos como checkout.
